Для запуска необходимо чтобы была установлена JRE (Java Runtime Environment)

Вариант 1:
После устновки JRE, необходимо добавить в переменную среды PATH путь к файлам JRE (C:\Program Files\Java\jre7\bin).

Вариант 2:
Открыть в блокноте файл run.bat и заменить строку "java -cp Graph.jar org.dyndns.phpusr.graph.Main" на строку вида "C:\Program Files\Java\jre7\bin\java -cp Graph.jar org.dyndns.phpusr.graph.Main", где "C:\Program Files\Java\jre7\bin\" путь к вашей JRE (может отличаться).

Запускать файл run.bat
